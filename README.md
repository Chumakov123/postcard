# Postcard
Сервис для создания простых веб-открыток. Позволяет сгенерировать ссылку с поздравлением, сопровождающимся визуальными эффектами, музыкой, фото-розыгрышем получателя.

**Демо**: [chumakov123.github.io/postcard](https://chumakov123.github.io/postcard)

<img width="813" height="503" alt="image" src="https://github.com/user-attachments/assets/9b5cc873-8c69-42d6-a089-b86b2510b898" />

## Возможности

- **Текстовое поздравление** — позволяет добавить к открытке текст.
- **Визуальные эффекты** — падающие звезды, фейерверки или конфетти на странице.
- **Музыка** — воспроизводит mp3 файлы по прямой ссылке. Из-за CORS политики браузеров не все источники доступны.
- **Фото‑розыгрыш** — с разрешения пользователя делает селфи и добавляет к нему клоунский нос. Изображения нигде не хранятся, обрабатываются локально при помощи face-api.
